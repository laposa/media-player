# Laposa Media Player

Play media over local network on Android TV and Amazon Fire TV.

## Features

- Small size of 30MB.
- No growing in size with usage - saves your storage.
- Supports NFS, SMB, SFTP.
- Clutter free UI.

## Technologies Used

- Kotlin
- Embrace (OpenTelemetry)
- Firebase Crashlytics
